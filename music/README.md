## 1. Demo简介

本demo基于SpringBoot+MySQL的数据库读取音乐播放的网站。

![QQ截图20210513182905](https://tuyong.oss-cn-hangzhou.aliyuncs.com/img/20210513184012.png)

### 1.1 技术应用

* MySQL
* SpringBoot
* druid

### 1.2 数据库

数据库中存放一个用户admin：密码：123

库中音乐一部分存放在本地，一部分存在云端
