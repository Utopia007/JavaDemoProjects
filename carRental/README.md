## 1. Demo简介

本demo基于SSM+LayUI的汽车租赁管理系统，实现汽车信息发布与用户租赁汽车的功能

### 1.1 技术实现

* Spring
* SpringMVC
* MyBatis
* LayUI
* MySQL
* Jsp
* pagehelper
